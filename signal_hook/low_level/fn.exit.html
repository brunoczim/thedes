pub fn exit(status: c_int) -> !
Expand description

A bare libc exit.

Unlike the std::process::exit, this one is guaranteed to contain no additions or wrappers and therefore is async-signal-safe. You can use this to terminate the application from within a signal handler.

Also, see register_conditional_shutdown.
